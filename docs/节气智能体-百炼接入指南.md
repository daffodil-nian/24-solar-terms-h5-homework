# 节气智能体 · 阿里云百炼接入指南

> 本网站首页已预留「节气智能体」对话 UI（`js/ai-agent.js`）。  
> 推荐接入 **阿里云百炼 · 智能体应用**，与静态 HTML 站点配合良好。

---

## 一、为什么推荐百炼智能体

| 对比项 | 阿里云百炼智能体 | 说明 |
|--------|------------------|------|
| 中文与节气场景 | ✅ 很适合 | 通义千问系列中文表现好 |
| 知识库 RAG | ✅ 支持 | 可上传 `docs/`、`term-food-data.js` 整理成的文档 |
| 零代码试用 | ✅ 控制台即可 | 先调 Prompt，再发布 |
| 嵌入网站 | ✅ 多种方式 | AppFlow 挂件 / API + 函数计算代理 |
| 费用 | 新用户有免费额度 | 按 Token 计费，课程/作品量级通常可控 |

**建议选择：** 百炼控制台 → **应用** → **智能体应用**（单 Agent），不要用「工作流」除非你要复杂分支。

官方文档：

- [智能体应用](https://help.aliyun.com/zh/model-studio/user-guide/single-agent-application)
- [智能体应用 API](https://help.aliyun.com/zh/model-studio/new-agent-application-api-reference)
- [10 分钟给网站加 AI 助手（AppFlow）](https://help.aliyun.com/zh/model-studio/add-an-ai-assistant-to-your-website-in-10-minutes)

## 二、在百炼里创建「节气智能体」（控制台）

### 1. 开通与密钥

1. 登录 [百炼控制台](https://bailian.console.aliyun.com/)
2. 开通模型服务（新用户一般有免费额度）
3. [API Key 管理](https://bailian.console.aliyun.com/?tab=model#/api-key) → **创建 API-KEY**，保存到本地（勿提交到 Git）

### 2. 创建智能体应用

1. 控制台 → **应用** → **创建应用** → 选择 **智能体应用**
2. **模型**：`通义千问-Plus` 或 `Qwen3.5-Plus`（平衡成本与效果）
3. **系统 Prompt 示例**（可粘贴后微调）：

```
你是「二十四节气」网站专属助手，名叫「节气小助」。

职责：
- 回答二十四节气的日期、物候、习俗、诗词、饮食、养生
- 语气古朴简洁，像语文课本里的讲解，不要编造不存在的民俗
- 不确定时说明「建议查阅权威资料」，并提示用户看站内对应节气页

回答要求：
- 优先结合中国南北差异说明
- 单次回答控制在 200～400 字，除非用户要求详细展开
- 不要输出与节气无关的内容
```

4. **知识库（强烈建议）**  
   - 控制台 → **知识库** → 上传文档  
   - 可上传：`docs/内容替换指南.md`、各节气文案导出、你自己整理的 Word/PDF  
   - 在智能体里 **启用 RAG / 知识库**，减少胡编

5. 右侧 **测试** 通过后，点击右上角 **发布**

### 3. 记录两个 ID

| 名称 | 在哪里看 |
|------|----------|
| **应用 ID（APP_ID）** | 应用管理 → 应用详情 |
| **API Key** | 密钥管理（sk- 开头） |

## 三、接到本网站：三种方式（选一种）

### 方式 A · 最快：AppFlow 悬浮挂件（几乎不改代码）

适合：先演示、不想写后端。

1. 按官方教程 [10 分钟接入](https://help.aliyun.com/zh/model-studio/add-an-ai-assistant-to-your-website-in-10-minutes)  
2. 百炼创建应用 → **AppFlow** 创建 AI 助手 → **Web 页面集成**  
3. 复制 **悬浮挂件脚本**，粘贴到 `html/index.html` 的 `</body>` 前  

**优点：** API Key 不在你前端代码里，由 AppFlow 托管。  
**缺点：** 样式是 AppFlow 默认挂件，与站内 `ai-agent-panel` 不完全一致（可保留两套，或隐藏 FAB 只用 AppFlow）。

---

### 方式 B · 推荐：保留现有 UI + 函数计算代理（安全）

适合：继续使用本站右下角 **✦ 节气智能体** 面板。

**原因：** 浏览器里 **不能直接写百炼 API Key**（会泄露，且常有跨域限制）。  
需要加一个 **后端转发**（推荐 [函数计算 FC](https://fcnext.console.aliyun.com/) 或任意 Node 小服务）。

#### 架构

```
用户浏览器 (ai-agent.js)
    → POST /api/chat  { message: "雨水吃什么" }
        → 函数计算（保存 API Key）
            → POST https://dashscope.aliyuncs.com/api/v1/apps/{APP_ID}/completion
    ← 返回 { reply: "……" }
```

#### 函数计算示例（Node.js）

在 FC 创建 HTTP 函数，环境变量：

- `DASHSCOPE_API_KEY` = 你的 sk-xxx  
- `BAILIAN_APP_ID` = 智能体应用 ID  

```javascript
// index.js（函数计算 handler 示例）
const https = require('https');

exports.handler = async (event, context) => {
  const body = JSON.parse(event.body || '{}');
  const message = (body.message || '').trim();
  if (!message) {
    return { statusCode: 400, headers: cors(), body: JSON.stringify({ error: 'empty message' }) };
  }

  const payload = JSON.stringify({
    input: { prompt: message },
    parameters: {}
  });

  const appId = process.env.BAILIAN_APP_ID;
  const apiKey = process.env.DASHSCOPE_API_KEY;

  const res = await new Promise((resolve, reject) => {
    const req = https.request({
      hostname: 'dashscope.aliyuncs.com',
      path: `/api/v1/apps/${appId}/completion`,
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer ' + apiKey,
        'Content-Length': Buffer.byteLength(payload)
      }
    }, (r) => {
      let data = '';
      r.on('data', (c) => { data += c; });
      r.on('end', () => resolve({ status: r.statusCode, data }));
    });
    req.on('error', reject);
    req.write(payload);
    req.end();
  });

  let json;
  try { json = JSON.parse(res.data); } catch (e) {
    return { statusCode: 502, headers: cors(), body: JSON.stringify({ error: 'bad upstream' }) };
  }

  if (res.status !== 200) {
    return { statusCode: res.status, headers: cors(), body: JSON.stringify({ error: json.message || 'upstream error' }) };
  }

  return {
    statusCode: 200,
    headers: cors(),
    body: JSON.stringify({ reply: json.output && json.output.text ? json.output.text : '' })
  };
};

function cors() {
  return {
    'Content-Type': 'application/json',
    'Access-Control-Allow-Origin': '*', // 上线时改成你的域名
    'Access-Control-Allow-Headers': 'Content-Type'
  };
}
```

#### 前端配置（本站）

1. 复制 `js/ai-agent-config.example.js` → `js/ai-agent-config.js`（**不要提交 API Key 到 Git**）
2. 填写你的 FC 公网地址：

```javascript
window.TermAIAgentConfig = {
  apiUrl: 'https://你的FC域名/api/chat',
  parseResponse: function (data) {
    return data.reply || data.output?.text || '暂无回复';
  }
};
```

3. 在 `html/index.html` 中，在 `ai-agent.js` **之前** 引入：

```html
<script src="../js/ai-agent-config.js"></script>
<script src="../js/ai-agent.js"></script>
```

4. 刷新首页，右下角 ✦ 即可走真实智能体（不再显示「演示模式」）

**百炼官方 API 地址：**  
`POST https://dashscope.aliyuncs.com/api/v1/apps/{APP_ID}/completion`  
请求体：`{ "input": { "prompt": "用户问题" } }`  
响应正文：`output.text` 为回复内容。

多轮对话：保存 `output.session_id`，下次请求带上 `input.session_id`（需在 FC 里扩展，前端传 sessionId）。

### 方式 C · 官方网页分享（仅演示）

1. 智能体 **发布** → **发布渠道** → **官方网页版** → 创建链接  
2. 把链接放在首页按钮「打开智能体」  

**注意：** 链接通常要求访问者是阿里云账号用户，且链接有过期时间，**不适合作为正式公众网站方案**。

## 四、与本项目文件的对应关系

| 文件 | 作用 |
|------|------|
| `js/ai-agent.js` | 对话 UI + `fetch` 调用 `apiUrl` |
| `js/ai-agent-config.js` | **你本地创建**，填代理地址（勿提交密钥） |

未配置 `apiUrl` 时，`ai-agent.js` 运行在 **演示模式**（随机节气话术）。

## 六、费用与安全提醒

- **API Key 只放在服务器环境变量**，不要写在 `ai-agent.js` 或提交 Git  
- 将 `ai-agent-config.js` 加入 `.gitignore`  
- 函数计算设置 **HTTP 鉴权** 或 **Referer 限制**，防止接口被刷  
- 在百炼控制台查看 **Token 用量**，设置预算告警
