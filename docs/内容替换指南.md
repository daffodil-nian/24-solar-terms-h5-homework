# 二十四节气网站 · 内容替换指南

> 本文档说明：**哪个页面、哪个区域**可以替换**图片、文字、音频/视频**，以及对应文件路径。  
> 多媒体命名细则另见 [`节气专题多媒体交付说明.md`](./节气专题多媒体交付说明.md)。

## 一、先看这张总表

| 内容类型 | 主要存放位置 | 主要配置文件 |
|----------|--------------|--------------|
| **节气专题页** 诗词 / 美食 / 音频 / 视频 | `images/terms/`、`media/terms/` | `js/term-pages-data.js` |
| **节气专题页** 主题色 / 题签 / 背景动画类型 | — | `js/term-themes.js` |
| **栏目页** 诗词书画、民俗美食中的节气卡片 | `images/terms/` | `html/poetry.html`、`html/customs.html` + `js/term-food-data.js` |
| **全站导航栏** 各页标题与段落 | 各 `.html` 文件内 | 直接改 HTML |
| **全站背景音乐** | `media/`（`bg_music1.mp3` 等） | `js/site-bgm-config.js` |
| **四时之约 / 联系 / 地理** 等专题 | 见下文各节 | 各专题 `*-config.js` 或 HTML |

**slug 对照**（中文名 → 文件名前缀）：见 `js/term-slugs.js`  
例：雨水 `yushui`、谷雨 `guyu`、清明 `qingming`。

## 二、24 个节气专题页（最重要）

### 2.1 页面文件

每个节气一个独立 HTML：

```
html/terms/{slug}.html
```

例：`html/terms/yushui.html`（雨水）、`html/terms/guyu.html`（谷雨）。

也可从旧链接进入：`html/term.html?t=yushui`（会自动跳转到 `terms/yushui.html`）。

### 2.2 页面结构（从上到下）

| 区域 | 锚点 | 显示内容 | 改文字 | 改图片 | 改音频/视频 |
|------|------|----------|--------|--------|-------------|
| Hero 标题区 | — | 节气名、日期、简介 | ✅ | ❌ | ❌ |
| 诗词书画 | `#poetry` | 诗句、文献、书画说明、插图 | ✅ | ✅ | ❌ |
| 民俗美食 | `#food` | 食俗、做法、配图、食俗诗 | ✅ | ✅ | ❌ |
| 节令音频 | `#audio` | 1 段可点击播放的音频 | ✅ | ❌ | ✅ |
| 节气影像 | `#media` | 2 个视频播放器 | ✅ | ❌ | ✅ |
| 全屏背景动画 | — | 雨、花、鸟等（不可点击） | ❌ | ❌ | ❌ |

**全部文案与媒体路径集中在：`js/term-pages-data.js`**

### 2.3 `term-pages-data.js` 字段说明（以雨水为例）

```javascript
yushui: {
  name: '雨水',                    // ← 改显示名
  date: '2月18—20日',              // ← 改日期
  heroTitle: '雨水 · 润物无声',     // ← Hero 大标题
  heroDesc: '……',                  // ← Hero 简介段落

  poetry: {
    title: '幽兰 · 润物 · 三候',   // ← 诗词区标题
    quote: '……',                   // ← 引文
    cite: '……',                    // ← 出处
    prose: '……',                   // ← 正文
    artNote: '……',                 // ← 书画说明
    poemDu: { text, cite },        // ← 可选：额外诗句块
    sanhou: '……',                  // ← 可选：三候
    illustration: {                 // ← 诗词区配图
      src: 'orchid-rain.svg',      //   文件名，放 images/terms/
      alt: '……',
      caption: '……'
    }
  },

  food: {
    title: '……',                   // ← 美食区标题
    intro: '……',                   // ← 导语
    items: [{ title, desc, recipe }], // ← 菜品卡片
    folk: '……',                    // ← 民俗说明
    poem: { verse, author },        // ← 食俗诗词
    images: [{ src, alt, role }]   // ← 美食配图（images/terms/）
  },

  audio: {
    title: '……',
    subtitle: '……',
    desc: '……',
    src: 'yushui.mp3'              // ← 放 media/terms/yushui.mp3
  },

  videos: [
    { title, desc, src: 'yushui-1.mp4' },  // ← media/terms/
    { title, desc, src: 'yushui-2.mp4' }
  ]
}
```

**其余 20 个节气**（无单独详细块）由同文件末尾脚本自动生成，并合并 `js/term-food-data.js` 中的美食与诗词。

### 2.4 替换图片（节气专题）

| 用途 | 放哪里 | 在 data 里怎么写 |
|------|--------|------------------|
| 诗词区插图 | `images/terms/` | `poetry.illustration.src` |
| 美食配图 | `images/terms/` | `food.images[].src` |
| 视频封面（可选） | `images/terms/` | `videos[].poster`（不写则用 `{slug}.png`） |
| 栏目网格缩略图 | `images/terms/{slug}.png` 或 `.svg` | 无需改 data，见 `js/term-images.js` |

**雨水相关现有 SVG 示例：**

| 文件 | 用途 |
|------|------|
| `orchid-rain.svg` | 雨水诗词插图、诗词书画页卡片 |
| `jicai.svg` | 雨水美食（荠菜） |
| `yushui.svg` | 雨水节气卡片主图 |
| `yushui-ink-bg.svg` | 雨水页背景水墨（CSS 引用，见 `css/yushui.css`） |

### 2.5 替换音频 / 视频（节气专题）

```
media/terms/
├── yushui.mp3          ← 音频：data.audio.src
├── yushui-1.mp4        ← 视频一：data.videos[0].src
├── yushui-2.mp4        ← 视频二：data.videos[1].src
├── guyu.mp3
├── guyu-1.mp4
└── …
```

改文件名时，**必须同步修改** `term-pages-data.js` 中对应 `src` 字段。

### 2.6 背景动画（不改 HTML，改配置）

| 节气 | 动画类型 | 配置文件 |
|------|----------|----------|
| 立春 | 梅花飘落 | `js/plum-blossom-fx.js` + `term-themes.js` → `fx: 'plum'` |
| 雨水 | 江南细雨 + 露珠 | `js/term-fx.js` + `css/yushui.css` → `fx: 'rain'` |
| 清明 | 燕鸟 + 草芽 | `js/qingming-fx.js` → `fx: 'qingming'` |
| 谷雨 | **密集降水** | `js/guyu-fx.js` → `fx: 'guyu'` |
| 其余 | 通用动效 | `js/term-themes.js` 各节气 `fx` 字段 |

修改后若需重建 HTML 壳：`node scripts/generate-term-html.js`

## 七、替换操作速查

### 只换「节气美食专题」大册里的美食图

1. 将美食照片放入 **`images/food/`**，推荐命名：`lichun.jpg`、`yushui.jpg` …（slug 见 `js/term-slugs.js`）
2. 打开 **`js/term-food-data.js`**，在对应节气下设置：
   ```javascript
   '雨水': {
     …
     image: 'food/yushui.jpg',   // 或 'jicai.svg'（images/terms/ 下）
     imageAlt: '荠菜馄饨',
   }
   ```
3. 刷新 `html/customs.html` 的「节气美食专题」区块

也可写完整路径：`image: '../images/food/my-photo.webp'`  
未配置 `image` 时，会回退到 `images/terms/` 下的美食 SVG 或季节底图。

### 只换雨水专题页的一张美食图

1. 将新图放入 `images/terms/`，如 `jicai-new.jpg`
2. 打开 `js/term-pages-data.js` → `yushui.food.images` 中把 `src` 改为 `jicai-new.jpg`
3. 刷新 `html/terms/yushui.html#food`

### 只换雨水音频

1. 替换或新增 `media/terms/yushui.mp3`
2. （可选）改 `term-pages-data.js` → `yushui.audio` 的 `title`、`desc`
3. 刷新 `#audio` 区域

### 只换诗词书画页雨水卡片图

1. 替换 `images/terms/orchid-rain.svg`（或新文件名）
2. 若改名，改 `html/poetry.html` 中两处 `src="../images/terms/……"`

### 换谷雨视频并加强雨效

1. 视频：`media/terms/guyu-1.mp4`、`guyu-2.mp4`
2. 文案：`term-pages-data.js` 中 `guyu` 块（或自动生成段的 `videos`）
3. 雨量密度：`js/guyu-fx.js` → `fineCount` / `midCount` / `heavyCount`
