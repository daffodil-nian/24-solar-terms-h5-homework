这是一个前端课程大作业，国风二十四节气展示网页，包含节气图文、季节轮播、时间观象模块，完整可运行H5页面。

项目类型	中国传统文化主题网站（静态前端 + 轻量后端代理）
选题方向	非物质文化遗产数字化传播

 选题策划思路  
 选题缘起  
二十四节气是中华民族独有的时间知识体系，将天文、气象、物候、农事、民俗融为一体，被誉为 "中国第五大发明"。然而在数字时代，年轻一代对节气的认知往往停留在手机日历上几个陌生的名词 ——"惊蛰"" 芒种 ""处暑" 的物候意象、背后的诗词典故、各地的饮食民俗，正在从日常记忆中淡去。
本项目的核心命题是：能否用当代前端技术，为二十四节气创造一个「克制而有质感」的数字栖息地？
 设计理念 —— 克制与留白  
作品在视觉层面刻意回避了 "热闹的国潮风"，走的是东方留白美学路线：
1. 色彩体系：以青绿（teal/cyan）为主色调，象征 "天地气韵流转"；四季各有对应色 —— 春粉、夏绿、秋金、冬灰，节气专题页则对应每个节气的物候意象定制配色（如立春的寒梅粉白、雨水的烟雨青灰）。
2. 版式语言：大量留白、宽屏容器（1440px）、字体的行距与字间距经过反复调试，让内容 "有呼吸感"。
3. 动画选择：极光背景、梅瓣飘落、雨丝涟漪、流萤飞舞 —— 所有动效都服务于节气氛围，不做炫技式动效堆砌。

● 节气专属粒子特效系统：摒弃全站统一静态背景，为 24 个节气配置 18 种独立粒子动画，通过 HTML 自定义属性一键切换视觉氛围。

 核心功能详解  
 智能节气定位  
系统读取设备本地日期，自动匹配当前所处节气，首页实时展示：
● 今日节气标签：节气名称、公历日期区间
● 时令诗词横幅：匹配节气古典诗词 + 白话文释义
● 三候物候卡片：完整展示该节气 72 候对应自然现象
● 下一个节气倒计时天数展示
 AI 节气智能体  
页面右下角浮动星标按钮唤起对话面板，支持节气全场景问答；
● 联网配置阿里云百炼 API：大模型实时精准解答民俗、养生、农事问题；
● 离线无密钥环境：自动切换演示模式，内置节气相关模拟问答逻辑； 支持一键关闭对话弹窗，不遮挡页面浏览。
 知识挑战闯关系统  
 题库共 50 道，覆盖地理、民俗、诗词、养生、历法五大分类； 交互流程：选择分类→连续作答 10 题→实时判分展示解析→结束生成节气达人等级评价； 配套题典浏览模式，可无限制查阅全部题目标准答案。  
 民俗抽签翻书交互  
 模拟传统抽签仪式动画流程：摇签→抽取节气签条→古书翻页展示三层内容（封面 / 民俗典故 / 完整食谱）； 全局支持键盘左右方向键、页面箭头控制翻页；配套完整 24 节气全景翻书阅览模块。  
 全局背景音乐系统  
 进入首页自动循环播放国风背景音乐，右下角音符按钮控制播放暂停； 采用sessionStorage存储播放状态，跨页面持续播放； 「四时之约」页面自动暂停全局背景音乐，避免与页面专属主题音频冲突。  
 独立节气专题页  
每个节气专属页面四大内容板块：
● 诗词书画区：节气专属古诗词、名家书画赏析；
● 民俗美食区：节气传统饮食、完整家常制作步骤；
● 节气影像区：前 9 个节气双主题短片，后 15 节气配套自然环境音频；
● 节令音频区：节气专属自然白噪音 / 国风配乐； 页面搭载对应节气专属粒子动态背景，视觉与节气意象统一。

 核心技术方案  
 数据驱动批量生成节气页面  
 统一数据源文件term-pages-data.js存储全部节气文本素材，通过数组遍历匹配节气标识批量填充页面模板；配套generate-term-html.js脚本一键生成全部节气 HTML 文件。优势：新增、修改节气内容仅修改单一数据文件，无需修改页面结构，维护工作量降低 90% 以上。  
 CSS 变量全局主题配色系统  
term-themes.js为每个节气预定义 5 组主题色变量，通过 JS 动态挂载至根节点，全局 CSS 统一调用变量渲染页面色彩。  
/* 节气页面配色引用示例 */
.term-hero { background: var(--t-bg1); }
.term-hero h1 { color: var(--t-ink); }
 Canvas 粒子特效调度系统  
term-fx.js内置 18 套节气专属粒子动画，分为手工精写特效（梅花、春雨、飞燕等 4 套）与通用动态特效（落叶、雾、萤火、飞雪等 14 套）；通过页面data-fx属性自动匹配渲染；动画采用 GPU 加速 transform 渲染，pointer-events: none不干扰页面点击交互。  

 兼容降级多媒体方案  

● 视频：原生 Video 标签 + 自定义 JS 进度控件，设置封面兜底图，资源加载失败自动隐藏视频模块；
● 音频：背景音乐与节气独立音频分离管理，会话存储记录播放状态；
● 图片：四级资源回退逻辑，缺失图片自动匹配季节兜底矢量图，无资源白屏问题。
 前端性能优化策略  

● 页面切换使用 CSS view-transition 平滑过渡；
● 音视频设置preload="metadata"，不阻塞 DOM 渲染；
● 粒子动画使用requestAnimationFrame节流降低设备负载；
● 全局加载缓冲屏，固定 1.4s 视觉缓冲，防止内容闪烁。
 项目五大创新点  
● 时间感知动态首页：网站实时读取本地日期自动匹配当日节气，首页诗词、物候、倒计时全部动态刷新，网站具备实时时间生命力；
● 节气意象专属视觉动画：摒弃统一背景，每个节气匹配对应自然粒子特效，动画本身作为传统文化视觉表达载体；
● 多维立体非遗内容架构：单一节气串联历法历史、诗词、饮食、养生、现代文艺五大维度，完整构建立体文化认知，信息不割裂；
● 场景化 AI 节气助手：AI 并非单纯展示功能，贴合用户真实需求解决时令养生、民俗疑问，与节气网站定位深度融合；
● 轻量化可扩展低维护架构：数据与页面分离、CSS 变量主题化、脚本批量生成页面，单人即可维护超 35 页面大型文化网站。
