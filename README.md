## 一、项目基本信息

**项目名称**：二十四节气 · 观象授时

**项目类型**：中国传统文化主题网站（静态前端 + 轻量后端代理）

**选题方向**：非物质文化遗产数字化传播

**对应非遗**：二十四节气（2016 年入选联合国教科文组织人类非物质文化遗产代表作名录）

**技术栈**：

- 前端：HTML5 + CSS3 + JavaScript（jQuery + Bootstrap 5）
- 后端代理：Node.js（阿里云函数计算）
- AI 对话：阿里云百炼 API
- 多媒体：HTML5 Video/Audio、SVG 矢量动画、Canvas 粒子特效

**项目规模**：
- 页面数：35+ 个 HTML 页面（首页 ×2、节气专题 ×24、功能页面 ×9）
- JS 模块：35+ 个独立脚本文件
- CSS 样式表：25+ 个主题与功能样式表
- 媒体资源：50+ 个视频/音频文件、80+ 张节气与美食图片
- 数据集：50 道知识挑战题、24 节气 ×（诗词 + 物候 + 美食 + 民俗）完整内容

**运行环境**：任意支持 HTML5 的现代浏览器，通过本地文件或任意静态服务器即可运行。

## 二、选题策划思路

### 2.1 选题缘起

二十四节气是中华民族独有的时间知识体系，将天文、气象、物候、农事、民俗融为一体，被誉为"中国第五大发明"。然而在数字时代，年轻一代对节气的认知往往停留在手机日历上几个陌生的名词——"惊蛰""芒种""处暑"的物候意象、背后的诗词典故、各地的饮食民俗，正在从日常记忆中淡去。

本项目的核心命题是：**能否用当代前端技术，为二十四节气创造一个「克制而有质感」的数字栖息地？**

### 2.2 设计理念——克制与留白

作品在视觉层面刻意回避了"热闹的国潮风"，走的是**东方留白美学**路线：

- **色彩体系**：以青绿（teal/cyan）为主色调，象征"天地气韵流转"；四季各有对应色——春粉、夏绿、秋金、冬灰，节气专题页则对应每个节气的物候意象定制配色（如立春的寒梅粉白、雨水的烟雨青灰）。
- **版式语言**：大量留白、宽屏容器（1440px）、字体的行距与字间距经过反复调试，让内容"有呼吸感"。
- **动画选择**：极光背景、梅瓣飘落、雨丝涟漪、流萤飞舞——所有动效都服务于节气氛围，不做炫技式动效堆砌。

### 2.3 用户定位与场景

- **主要人群**：对中国传统文化有兴趣的年轻网民、在校学生、文化教育工作者
- **核心场景**：
  - 日常浏览——首页根据当天日期自动切换到对应节气，展示诗词与物候
  - 深度学习——知识页按节气逐一展开，从历史源流到地理分布
  - 趣味互动——抽签翻书、答题闯关、AI 对话
  - 教学辅助——日历总表、健康养生指南、文化联动库

### 2.4 差异化策略

| 维度 | 同类网站常见做法 | 本作品的取舍 |
|------|----------------|-------------|
| 视觉风格 | 大红色、灯笼、国潮插画 | 留白、克制、物候色系 |
| 内容深度 | 一句话介绍每个节气 | 诗词 + 三候 + 美食 + 民俗 × 24 |
| 多媒体 | 一篇图文到底 | 每个节气 2 段专题视频 + 1 段音频 |
| 交互 | 点击翻页 | AI 节气智能体、知识挑战闯关、翻书抽签 |
| 个性化 | 无 | 进入首页自动定位"今天的节气" |

## 三、项目开发过程

### 3.1 总体时间线

| 阶段 | 主要工作 | 关键产出 |
|------|---------|---------|
| **第一阶段 · 基础搭建** | 确定技术栈（jQuery + Bootstrap 5）、搭建页面骨架、创建 navbar 导航体系 | `index.html`、`main.css`、导航框架 |
| **第二阶段 · 内容填充** | 编写 24 节气数据模块、诗词/物候/美食/民俗内容录入、创建 24 个节气详情页 | `terms-data.js`、`terms-seasonal-content.js`、`term-food-data.js`、24 个 `html/terms/*.html` |
| **第三阶段 · 多媒体整合** | 录制/搜集节气视频与音频、编写自定义播放器控件、管理 50+ 媒体文件 | `video.js`、`media/terms/` 目录、背景音乐系统 |
| **第四阶段 · 视觉升级** | 节气专题页视觉主题系统、极光背景、背景粒子特效（雨、梅、萤等）、滚动揭示动画 | `term-themes.js`、`aurora.css`、`term-fx.js`、`scroll.js` |
| **第五阶段 · 功能深化** | AI 节气智能体、知识挑战问答系统、抽签翻书交互、文化联动库 | `ai-agent.js`、`challenge.js`、`customs-food-draw.js`、`culture-library.js` |
| **第六阶段 · 适配打磨** | 响应式布局、加载动画、节气日历、健康养生页、四时之约页、页面跳转过渡 | `responsive.css`、`site-loader.js`、`calendar.js`、`health.js`、`heritage.js` |

### 3.2 迭代中的关键决策

1. **从"一套首页"到"双首页并存"**：最初只有一个传统布局的首页，后来追加了 Premium 版首页（极光 + 卡片导航 + 倒计时），两个首页通过 `index.html` 和 `index2.html` 共存，用户可根据喜好选择入口。

2. **从"手动写 24 个页面"到"脚本生成 + 数据驱动"**：前 4 个节气专题页是手工精写的（立春、雨水、惊蛰、春分），其余 20 个通过 `generate-term-html.js` 脚本自动生成骨架，内容由 `term-pages-data.js` 中的数据统一驱动，大幅降低了维护成本。

3. **从"单一背景"到"节气专属特效系统"**：最初所有节气页共用背景，后来为每个节气设计了独立的 `data-fx` 粒子特效（梅、雨、雷、燕、萤、竹、枫、雪等 18+ 种），通过 CSS 自定义属性实现主题色的无缝切换。

4. **AI 对话的实用化**：从最初的纯 ChatGPT 占位，到接入阿里云百炼 API，再到支持"演示模式"（无需 API Key 即可体验），让项目在不同环境下都有可用的对话体验。

### 4.2 核心功能详解

#### 4.2.1 智能节气定位

系统根据电脑/手机的当前日期，自动判断处于哪个节气，并在首页展示：
- **今日节气芯片**：节气名称 + 日期范围
- **时令诗词横幅**：随节气动态切换的古典诗词 + 白话释义
- **三候物候卡**：该节气的三个物候现象（如立春：东风解冻→蜇虫始振→鱼陟负冰）
- **下一节气倒计时**：距离下个节气还有几天

#### 4.2.2 AI 节气智能体

右下角的 ✦ 按钮呼出 AI 对话框。用户可以向 AI 提问任何节气相关问题（如"今天适合吃什么""芒种有什么讲究"），AI 通过阿里云百炼大模型实时回答。未配置 API 时自动切换为演示模式。

#### 4.2.3 知识挑战闯关

50 道题目覆盖地理、民俗、诗词、养生、日历 5 个维度。答题流程：
1. 选择分类 → 2. 依次作答 10 题 → 3. 查看得分与等级评价 → 4. 每道题显示正确答案与解析

另有「题典模式」，可浏览全部题目及答案。

#### 4.2.4 民俗抽签翻书

模拟古时抽签仪式：摇签→出签→翻开三页古书，展示随机节气的食俗典故与食谱做法。另有 24 页通览翻书，可按左右箭头或←→键盘翻阅。

#### 4.2.5 背景音乐系统

进入首页自动播放多首背景音乐循环，右下角 ♫ 按钮可控制播放/暂停。状态由 sessionStorage 持久化，跨页面保持连续播放（仅在"四时之约"页面自动暂停，因为它有独立音频）。

#### 4.2.6 节气专题页

每个节气拥有独立专题页，包含：
- **诗词书画区**：节气主题诗词、古典名句引语、书画品赏笔记
- **民俗美食区**：2-3 道时令菜肴的典故与详细做法
- **节气影像区**：2 段主题短片（自夏至起仅显示音频）
- **节令音频区**：1 段节气配乐 / 自然声音景
- **专属视觉特效**：背景粒子动画，如立春梅花飘落、雨水雨丝涟漪、惊蛰闪电杏花、谷雨密集雨幕等

### 5.2 核心技术方案

#### 5.2.1 数据驱动的内容生成

24 个节气专题页通过单一数据源 `term-pages-data.js` 驱动。该文件对前 4 个节气提供精写内容，其余 20 个节气通过闭包内的自动生成逻辑（遍历 `solarTerms` 数组 → 匹配 slug → 填充通用模板）完成初始化。页面 HTML 由 `scripts/generate-term-html.js` 批量生成，保证了结构一致性。

**优点**：新增一个节气内容只需在数据文件中添加条目，无需修改任何 HTML，降低了 90% 以上的维护工作量。

#### 5.2.2 CSS 自定义属性主题系统

`term-themes.js` 为每个节气定义了 5 个颜色变量（背景、文字、强调色等），通过 `document.documentElement.style.setProperty()` 动态注入。节气页面的所有颜色引用这些 CSS 变量，一套 CSS 即可适配 24 套配色方案。

```css
/* 节气页面中的颜色引用 */
.term-hero { background: var(--t-bg1); }
.term-hero h1 { color: var(--t-ink); }
```

#### 5.2.3 粒子特效系统

`term-fx.js` 支持 18+ 种节气专属粒子动画，分为：
- **精写特效**（4 种）：立春梅、雨水露、清明燕、谷雨密集雨——手工编写的 SVG 动画，与节气意象高度绑定
- **通用特效**（14+ 种）：春风、雾雨、叶舞、萤火、竹影、秋枫、雪光等——通过 `initGenericTermFx()` 统一调度，根据 `data-fx` 属性自动匹配

所有特效使用 `position: fixed` + `pointer-events: none`，不影响页面交互，且通过 `will-change` 和 `transform` 实现 GPU 加速。

#### 5.2.4 响应式多媒体方案

- **视频**：使用原生 `<video>` 标签 + 自定义 JavaScript 控件（播放/暂停、静音、进度条拖动、时间显示），支持 poster 兜底图和加载失败降级
- **音频**：节令音频与背景音乐系统独立，通过 `sessionStorage` 跨页面保持播放状态
- **图片**：`term-images.js` 实现四级回退链（slug.png → slug.svg → 风物图 → 季节背景 SVG），确保任何缺失资源都有优雅兜底

#### 5.2.5 前端性能优化

- 页面过渡动画使用 `page-link-transition` 类 + CSS `view-transition`，实现平滑切换
- 视频设置 `preload="metadata"`，避免阻塞页面加载
- 背景粒子特效通过 `requestAnimationFrame` 节流
- 加载屏（`site-loader.js`）提供最小 1.4 秒的视觉缓冲，避免内容闪现

### 5.3 主要创新点

1. **"今日节气"动态感知**——首页不是静态展示 24 个节气，而是根据用户访问时的真实日期，自动定位当前节气并刷新所有关联内容（诗词/物候/倒计时），让网站具有"时间感"。

2. **节气专属视觉特效**——每个节气拥有独立的背景动画，不是简单的图片替换。立春的梅花飘落、雨水的雨丝涟漪、惊蛰的远雷杏花、谷雨的密集雨幕……特效本身即是节气意象的视觉翻译。

3. **多维度非遗内容呈现**——围绕同一个节气，同时展开"历法渊源—诗词书画—饮食民俗—养生智慧—影视联动"五个维度，形成立体的文化认知体验，而非割裂的信息罗列。

4. **AI 节气智能体的实用嵌入**——AI 不是"炫技插件"，而是帮助用户回答"今天适合做什么"这类真实问题的实用工具，与节气的时令属性天然契合。

5. **低维护成本的可扩展架构**——数据脚本分离、CSS 变量主题化、HTML 模板批量生成——三管齐下，让个人开发者也能维护一个 35+ 页面的复杂文化网站。

## 六、使用说明

### 6.1 本地运行

1. 将项目文件夹放置于任意目录。
2. 用浏览器直接打开 `index.html`（根目录）或 `html/index.html`。
3. 推荐使用 Chrome、Edge、Firefox 等现代浏览器，建议窗口宽度 ≥ 1200px 以获得最佳排版体验。
4. 无需安装任何依赖，无需启动本地服务器（纯静态 HTML 项目）。

### 6.3 节气专题页访问方式

- **从首页进入**：点击首页的「节气日历」→ 点击任意节气卡片
- **从知识页进入**：点击「节气知识」页面中 24 节气网格的任意节气
- **从诗词/美食页进入**：点击节气导航网格的任意节气
- **从诗词抽签进入**：点击「诗词书画」页的「随机诗卷」功能自动跳转

### 6.4 AI 节气智能体使用

1. 点击右下角 ✦ 浮动按钮，呼出对话面板。
2. 在输入框中输入问题（例如"今天是什么节气""芒种适合吃什么""小暑怎么养生"）。
3. 按回车或点击发送按钮，AI 将返回回答。
4. 点击 × 按钮关闭面板。
5. **演示模式**：若未配置阿里云百炼 API，系统自动降级为随机回复演示，仍可体验对话流程。

### 6.5 背景音乐控制

- 进入首页后背景音乐自动播放（浏览器可能拦截自动播放，点页面任意处即可启动）。
- 右下角 ♫ 按钮可随时暂停/继续播放。
- "四时之约"页面会自动暂停背景音乐，因为该页有独立音频内容。

### 6.6 知识挑战玩法

1. 进入「知识挑战」页面。
2. 选择答题分类（综合/地理/民俗/诗词/养生/日历）。
3. 依次作答 10 道题，每题选择后立即显示对错和解析。
4. 完成全部题目后查看得分和等级评价（如「节气达人」「略有小成」等）。
5. 点击「题典模式」可浏览全部 50 道题及其正确答案。

### 6.7 民俗抽签翻书玩法

1. 进入「民俗美食」页面。
2. 点击签筒中的签条进行摇晃抽签。
3. 抽中后显示节气名称。
4. 以翻书动画展示 3 页内容：封面 → 食俗典故 → 食谱做法。
5. 可通过 ←→ 箭头或键盘方向键翻页。
6. 下方「节气食俗通览」是完整的 24 页翻书，可按季节快速跳转。
